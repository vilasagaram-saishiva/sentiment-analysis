YouTube Text Data Analysis & Sentiment Modeling
Project Overview
This project performs a comprehensive exploratory data analysis (EDA) and Natural Language Processing (NLP) on YouTube comment data. The goal is to understand audience sentiment and how engagement rates (likes and replies) vary across different video categories like Entertainment, News, and Comedy.

Key Features
Data Cleaning: Handled over 298,000 raw records, including duplicate removal (88,979 rows) and null value treatment.

Sentiment Analysis: Leveraged TextBlob and NLTK to categorize user comments into positive, negative, and neutral sentiments.

Engagement Metrics: Calculated "Engagement Rate" as a function of views, likes, and comment counts to identify high-performing content niches.

Advanced Visualization: Created interactive bubble charts and scatter plots using Plotly to visualize audience behavior across 15+ categories.

Tech Stack
Languages: Python

Libraries: Pandas, NumPy (Data Manipulation), Matplotlib, Seaborn, Plotly (Visualization).

NLP Tools: NLTK, TextBlob.

Analysis Summary
Data Preprocessing: Standardized the dataset by removing special characters and handling non-UTF-8 encodings.

Categorical Insights: Analyzed engagement across diverse categories such as "People & Blogs," "News & Politics," "Gaming," and "Education".

Visualization:

Developed a Bubble Chart to map views against engagement rates, with marker sizes representing comment counts.Visualized distribution of audience engagement by category to identify which genres foster the most active communities.
